# Mozart > Prototypes > Prototype1

Δημιουργία Use Case 1: Υλοποίηση της λογικής με τα απολύτως απαραίτητα components:

- broker
- CEP
- rules engine (το οποίο σε πρώτη φάση θα είναι hardcoded business logic "μέσα" στο CEP)

Προτεινόμενες Τεχνολογίες:

- Kafka (για broker - ESB)
- ZooKeeper (για distributed coordination & orchestration)
- Spring Boot + Spring Framework (ως framework για υποστήριξη και rapid prototype development)
- Kafka Streams (για streaming client processing & pipelining)
- ksqlDB (για βάση δεδομένων σε stream processing περιβάλλον)
- Apache Flink (για stateful real time computations)
- Apache Spark (για distributed map/reduce)
- Confluent Schema Registry (για Schema Registry)
- Avro (για SerDe)
- Curator (για distributed state machine)
